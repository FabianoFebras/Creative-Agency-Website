# English

## Creative Agency Web

A simple website using only front-end concepts,part of the EBAC course **"Javascript Fullstack"**.The website simulates the site of an agency that works with web design and developing websites for others companies or people.
The main goal was to create a website fulli responsive for all sorts of devices.

### How does it Work

- The website has 5 sections which are "about",a form that the user can send his name and email for the company to return in the "contact" section that doesn't do anything because the focus is the responsive part,a "home" section,a "rating" section that simulates feedback of previous satisfieds customers and a "services" section that tells the user all the services the company can provide.

### Tecnologies

- HTML
- CSS
- JavaScript

### Demo

Open the `home.html` file in your browser to run the project.

# Português

## Site de Agência Criativa

Um site simples usando apenas conceitos de front-end, parte do curso EBAC **Javascript Fullstack**. O site simula o site de uma agência que trabalha com design e desenvolvimento web para outras empresas ou pessoas.
O objetivo principal era criar um site totalmente responsivo para todos os tipos de dispositivos.

### Como funciona

- O site possui 5 seções: "Sobre", um formulário onde o usuário pode enviar seu nome e e-mail para a empresa retornar na seção "Contato" (que não tem nenhuma função, pois o foco é a responsividade), uma seção "Página Inicial", uma seção "Avaliações" que simula o feedback de clientes satisfeitos e uma seção "Serviços" que lista todos os serviços que a empresa oferece.

### Tecnologias

- HTML
- CSS
- JavaScript

Abra o arquivo `home.html`no browser para rodar o projeto.
